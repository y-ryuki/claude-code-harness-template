# Code of Conduct

## 私たちの約束

オープンで歓迎的なコミュニティを育むため、コントリビューター・メンテナーとして、年齢・体格・障害・民族・性自認や性表現・経験レベル・国籍・容姿・人種・宗教・性的指向にかかわらず、すべての人にとってハラスメントのない参加体験を約束します。

## 私たちの基準

ポジティブな環境を作るために:

- 共感と思いやりを示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを行う
- 自分の過ちの責任を取り、影響を受けた人に謝罪する
- 個人だけでなくコミュニティ全体にとって最善を考える

許容されない行為:

- 性的な言葉・画像の使用、性的な誘い
- 荒らし、侮辱、軽蔑的なコメント、個人攻撃、政治的攻撃
- 公開・非公開のハラスメント
- 明示的な許可なく他者の個人情報を公開
- プロフェッショナルな場で不適切と判断されるその他の行為

## 報告

不適切な行為を見聞きした場合、リポオーナーに直接連絡してください。すべての苦情はレビューされ、適切に対応されます。

## 出典

この Code of Conduct は [Contributor Covenant](https://www.contributor-covenant.org/) version 2.1 に基づいています。
